　　　　　　　	　　　　テキストボックス拡張プラグイン

　ダウンロードいただき誠にありがとうございます
　ゲーム制作のお役に立てれば幸いです
　動作確認はティラノスクリプト v512a でおこなっています

――――――――――――――――――――――――――――――――――――――――――
■これは何ですか
――――――――――――――――――――――――――――――――――――――――――
　editタグ使用時に表示されるテキストボックスの外観を変更するプラグインです
　おまけとして入力内容の判定補助機能もつけてあります

――――――――――――――――――――――――――――――――――――――――――
■つかいかた
――――――――――――――――――――――――――――――――――――――――――
　プロジェクトファイルの data > others > plugin フォルダに「textbox_ex」フォルダを配置し
　シナリオファイル最初の方に[ plugin name="textbox_ex" ] と記述すると適用されます。
　使用できるマクロや関数につきましては init.ks のコメント欄をごらんください。
